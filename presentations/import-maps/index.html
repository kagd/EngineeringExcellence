What are import maps

  • A browser native mechanism to import dependencies at runtime
  • No longer bundle entire UI into a single chunk
  • Local libs and/or remote libs
  • Imports need to be ES modules
  • Shims are available for browsers that don't support all aspects of ES modules yet (Firefox is experimental)
  • Browser Support

Why are we using import maps?

  • Deploy shared libs across entire platform with a single merge
  • Independent deployment of features delivered by non-Online teams
  • Data Science with Emissions IQ
  • Limit innersourcing agreement duration
